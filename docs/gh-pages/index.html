System Requirements

  • OS: Ubuntu 20.04 LTS or later
  • Processor: Intel i5 / AMD Ryzen 5 (2.5 GHz+)
  • RAM: 4 GB minimum (8 GB recommended)
  • Storage: 500 MB free space
  • Audio: Built-in or USB audio device

Installation Instructions

# Download and install wget https://github.com/violet-dj/violet-dj-mixer/releases/download/v1.0.0/violet-dj-mixer_1.0.0_amd64.deb sudo apt install ./violet-dj-mixer_1.0.0_amd64.deb # Launch application violet-dj

Build from Source

git clone https://github.com/violet-dj/violet-dj-mixer.git cd violet-dj-mixer pip3 install -r requirements.txt python3 main.py
