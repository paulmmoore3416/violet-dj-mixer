System Specifications

System

  • Ubuntu 20.04 LTS+
  • Python 3.8+
  • PyQt6
  • Intel i5 / Ryzen 5 (min)
  • 4 GB RAM
  • 500 MB disk

  • PulseAudio (default)
  • ALSA (low-latency)
  • JACK (pro)
  • 44.1 / 48 / 96 kHz
  • 16 / 24 / 32-bit
  • <20ms latency

Connectivity

  • Bluetooth 4.0+ audio
  • Multi-device Bluetooth
  • Wi-Fi mDNS discovery
  • Network audio streaming
  • USB audio interfaces

  • FLAC / ALAC

  • WMA
